# User interface

## 3d viewport

**Overall**

`Mouse Mid` Orbiting

`Shift + Mouse Mid` Pan

> **gizmo** if no Mouse or Mid button, can use gizmo on Up-Right corner to Orbiting or Pan or Zoom etc.

`Shift + D ` Duplicate

`~` + `'View Selected'`  Focus
